**Windows Native. Local First. Open Source.**

Openza Tasks is the flagship Openza task manager for Windows. It stores your task data locally in SQLite, supports Markdown import/export, and can reconnect to Todoist and Microsoft To Do when you want provider sync.

## Quick Start

1. [Install Openza Tasks](/tasks/getting-started/installation) on Windows.
2. [Configure your task providers](/tasks/getting-started/configuration) if you want sync.
3. Add local tasks, import Markdown checkboxes, or sync provider tasks into Inbox review.
4. Keep local backups enabled, and optionally add OneDrive backup for device-loss recovery.
