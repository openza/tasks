# Welcome to Openza Tasks

Openza Tasks is a beautiful, distraction-free desktop application for managing your tasks from Todoist and Microsoft To-Do on Linux.

## Features

<CardGrid stagger>
  <Card title="Multi-Provider Support" icon="puzzle">
    Connect your Todoist or Microsoft To-Do accounts and manage all your tasks in one place.
  </Card>
  <Card title="4-Pane GTD Layout" icon="laptop">
    Productivity-focused interface with Navigation, Projects, Tasks, and Details panes.
  </Card>
  <Card title="Automatic Backups" icon="document">
    Daily automatic backups keep your data safe. Restore, download, or import backups anytime.
  </Card>
  <Card title="Dark Theme" icon="moon">
    Beautiful dark theme support for comfortable use in any lighting condition.
  </Card>
  <Card title="Import & Export" icon="random">
    Import tasks from markdown files and export your data for backup or sharing.
  </Card>
  <Card title="Native Performance" icon="rocket">
    Built with Flutter for smooth, responsive desktop performance. Quick to launch, easy on resources.
  </Card>
  <Card title="Secure by Design" icon="shield">
    OAuth 2.0 authentication with encrypted local storage. Your credentials stay safe.
  </Card>
  <Card title="Open Source" icon="github">
    MIT licensed and fully open source. Fork it, improve it, share it.
  </Card>
</CardGrid>
