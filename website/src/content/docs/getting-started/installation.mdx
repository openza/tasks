Openza Tasks is available for Linux (AppImage/Flatpak) and Windows (Installer).

## Download

<Tabs>
  <TabItem label="Windows">
    Download the latest installer from [GitHub Releases](https://github.com/openza/tasks/releases/latest).

    <Steps>
    1. Download `Openza-*-Setup.exe`
    2. Run the installer
    3. Follow the installation wizard
    4. Launch from Start Menu
    </Steps>

    **System Requirements:**
    - Windows 10 or later
    - x64 architecture
  </TabItem>

  <TabItem label="AppImage">
    Download the latest AppImage from the [GitHub Releases](https://github.com/openza/tasks/releases/latest) page.

    <Steps>
    1. Download the `.AppImage` file
    2. Make it executable:
       ```bash
       chmod +x Openza-*.AppImage
       ```
    3. Run the application:
       ```bash
       ./Openza-*.AppImage
       ```
    </Steps>

    **System Requirements:**
    - Ubuntu 22.04+, Fedora 35+, or Debian 12+
    - GLIBC 2.34 or newer
  </TabItem>

  <TabItem label="Flatpak">
    Download the Flatpak bundle from [GitHub Releases](https://github.com/openza/tasks/releases/latest).

    <Steps>
    1. Install Flatpak if not already installed:
       ```bash
       # Ubuntu/Debian
       sudo apt install flatpak

       # Fedora
       sudo dnf install flatpak
       ```
    2. Install the Openza Tasks Flatpak:
       ```bash
       flatpak install Openza-*.flatpak
       ```
    3. Run the application:
       ```bash
       flatpak run com.openza.tasks
       ```
    </Steps>
  </TabItem>
</Tabs>
