Openza Tasks V1 is Windows-first and Store-first.

## Microsoft Store

The public Microsoft Store listing is the preferred distribution channel.

- [Install Openza Tasks from Microsoft Store](https://apps.microsoft.com/detail/9NQGDKXGRGF8)
- [Open Openza Tasks in the Store app](ms-windows-store://pdp/?ProductId=9NQGDKXGRGF8)

## Local Developer Install

<Steps>
1. Install the WinUI/.NET toolchain:

   ```powershell
   winget configure -f https://aka.ms/winui-config
   ```

2. Open `Openza.Tasks.slnx` in Visual Studio.
3. Select the `Openza Tasks` launch profile.
4. Select `x64`.
5. Enable **Build** and **Deploy** for `Openza.Tasks` in Configuration Manager.
6. Press F5.
</Steps>

## Requirements

- Windows 10 22H2 or Windows 11
- x64 architecture for V1

Older GitHub releases are preserved for history, but the active product line is the Microsoft Store distributed Windows-native WinUI app.
