Openza Tasks works fully offline with local tasks. You can optionally connect providers for task sync and optionally connect OneDrive for cloud backup snapshots.

## Supported Providers

| Feature | Status | Auth Method |
|---------|--------|-------------|
| Local Tasks | Fully Supported | N/A (offline) |
| Todoist | Fully Supported | API Token |
| Microsoft To Do | Supported | Microsoft sign-in with MSAL.NET |
| OneDrive Backup | Optional | Microsoft sign-in with MSAL.NET |

## First Launch

When you first launch Openza Tasks, you can start using it immediately with local tasks. To sync with external providers, go to Settings > Integrations. To protect data beyond the current PC, go to Settings > Backups and enable OneDrive backup when you want it.

<Aside type="tip">
  Local tasks work completely offline and are stored on your device.
</Aside>

## Provider Setup Guides

- [Todoist Setup](/tasks/guides/todoist-setup) - Connect your Todoist account with an API token
- [Microsoft To Do Setup](/tasks/guides/mstodo-setup) - Choose the Microsoft account Openza should use for Microsoft To Do

## Settings

Access settings from the left navigation. Available sections include:

- **Appearance** - Toggle between light and dark themes
- **Integrations** - Connect or disconnect Todoist and Microsoft To Do
- **Spaces** - Manage local visibility scopes
- **Backups** - Configure local backups, optional OneDrive backup, and restore from previous backups

Import and export actions are available from the task command bar.

<Aside type="note">
Microsoft To Do and OneDrive backup each keep their own selected Microsoft account. They can use the same account or different accounts.
</Aside>
