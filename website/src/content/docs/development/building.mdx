Openza Tasks is built with .NET, Windows App SDK, and WinUI 3.

## Prerequisites

- Windows 10 22H2 or Windows 11
- .NET 10 SDK
- Visual Studio 2026 Community or newer with WinUI workloads

Install the recommended toolchain:

```powershell
winget configure -f https://aka.ms/winui-config
```

## Build Steps

<Steps>
1. Clone the repository:

   ```powershell
   git clone https://github.com/openza/tasks.git
   cd tasks
   ```

2. Restore packages:

   ```powershell
   dotnet restore Openza.Tasks.slnx
   ```

3. Run tests:

   ```powershell
   dotnet test src\Openza.Tasks.Tests\Openza.Tasks.Tests.csproj -c Release
   ```

4. Build the app:

   ```powershell
   dotnet build src\Openza.Tasks\Openza.Tasks.csproj -c Release --no-restore
   ```
</Steps>

## Visual Studio Debugging

Open `Openza.Tasks.slnx`, choose the `Openza Tasks` MSIX launch profile, select `x64`, enable Deploy for the app project, and press F5.
