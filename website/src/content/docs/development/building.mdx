Openza Tasks is built with Flutter and can be compiled for Linux, macOS, and Windows.

## Prerequisites

- Flutter SDK 3.7.0 or later
- Dart SDK 3.7.0 or later
- Rust toolchain (stable)
- Git

### Platform Dependencies

<Tabs>
  <TabItem label="Windows">
    - Visual Studio 2022 with "Desktop development with C++" workload
    - Install Rust: [rustup.rs](https://rustup.rs)
    - Install Inno Setup (for installer):
      ```powershell
      winget install JRSoftware.InnoSetup
      ```
  </TabItem>
  <TabItem label="Ubuntu/Debian">
    ```bash
    sudo apt install clang cmake ninja-build pkg-config \
      libgtk-3-dev libsecret-1-dev libjsoncpp-dev
    ```
  </TabItem>
  <TabItem label="Fedora">
    ```bash
    sudo dnf install clang cmake ninja-build \
      gtk3-devel libsecret-devel jsoncpp-devel
    ```
  </TabItem>
  <TabItem label="Arch">
    ```bash
    sudo pacman -S clang cmake ninja gtk3 libsecret jsoncpp
    ```
  </TabItem>
</Tabs>

### Rust Toolchain

Install Rust using rustup:

<Tabs>
  <TabItem label="Windows">
    Download and run [rustup-init.exe](https://rustup.rs)
  </TabItem>
  <TabItem label="Linux/macOS">
    ```bash
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    source ~/.cargo/env
    ```
  </TabItem>
</Tabs>

## Build Steps

<Tabs>
  <TabItem label="Windows">
    <Steps>
    1. **Clone the repository**
       ```powershell
       git clone https://github.com/openza/tasks.git
       cd tasks
       ```

    2. **Install dependencies**
       ```powershell
       flutter pub get
       ```

    3. **Generate code**
       ```powershell
       dart run build_runner build --delete-conflicting-outputs
       ```

    4. **Build Rust sync engine**
       ```powershell
       cd rust; cargo build --release; cd ..
       copy rust\target\release\openza_sync.dll .
       ```

    5. **Build the application**
       ```powershell
       flutter build windows --release
       ```

    6. **Create installer**
       ```powershell
       dart run inno_bundle --release --no-app
       ```
    </Steps>

    The installer will be in `build/windows/x64/installer/Release/`.
  </TabItem>

  <TabItem label="Linux">
    <Steps>
    1. **Clone the repository**
       ```bash
       git clone https://github.com/openza/tasks.git
       cd tasks
       ```

    2. **Install dependencies**
       ```bash
       flutter pub get
       ```

    3. **Generate code**
       ```bash
       dart run build_runner build --delete-conflicting-outputs
       ```

    4. **Build Rust sync engine**
       ```bash
       cd rust && cargo build --release && cd ..
       ```

    5. **Build the application**
       ```bash
       flutter build linux --release
       ```
    </Steps>

    The built application will be in `build/linux/x64/release/bundle/`.

    ### Creating an AppImage

    ```bash
    ./scripts/build-appimage.sh
    ```

    ### Creating a Flatpak

    ```bash
    ./scripts/build-flatpak.sh
    ```
  </TabItem>
</Tabs>
