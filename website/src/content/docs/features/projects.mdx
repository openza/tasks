Projects help you group related tasks together. Openza Tasks supports both local projects you create and read-only projects synced from external providers.

## Project Types

### Local Projects

Projects created in Openza Tasks that you fully control:

- Create, edit, and delete freely
- Assign a color
- Move tasks between them

### Provider Projects

Projects from Todoist or Microsoft To Do appear in the sidebar automatically:

- **Read-only** - Cannot edit or delete from Openza Tasks
- Managed through the original provider
- Tasks synced from these projects show their source project

<Aside type="note">
Provider projects are extracted from your synced tasks.
</Aside>

### The Inbox

Every integration has a default Inbox project:

- Always exists and cannot be deleted
- Tasks without a project go here
- Cannot be renamed or edited

## Creating a Project

<Steps>
1. Click **New project** in the command bar
2. Enter a project name
3. Click **Create**
</Steps>

The new project appears in your Projects pane under the Openza Tasks section.

## Editing a Project

<Steps>
1. Find the project in the Projects pane
2. Click **Edit**
3. Change the name, color, or favorite status
4. Click **Save**
</Steps>

<Aside type="note">
Only local Openza Tasks projects can be edited. Provider projects must be edited in their original app.
</Aside>

## Deleting a Project

<Steps>
1. Select the project
2. Click **Delete**
3. Choose what to do with the tasks:
   - **Move to Inbox** - Tasks are preserved in Inbox
   - **Delete tasks** - Tasks are permanently removed
4. Confirm the deletion
</Steps>

<Aside type="caution">
Deleting a project cannot be undone. Consider moving tasks to Inbox if you want to keep them.
</Aside>

## Moving Tasks Between Projects

You can move tasks to a different project in two ways:

### From Task Details

<Steps>
1. Select a task to open the details pane
2. Use the project dropdown to select a new project
3. Click **Save task**
</Steps>

## Filtering by Project

Click any project in the Projects pane to filter the task list:

- Only tasks from that project are shown
- The project name appears in the header
- Select **All projects** or another view to clear the filter

## Project Colors

When editing a local project, enter a hex color such as `#3b82f6`.

Colors are stored with the project and can be used by richer project visuals as the WinUI app matures.
