title: Labels
description: Categorize tasks with labels
---

Labels (called "tags" in some providers) allow you to categorize tasks across different projects.

## Viewing Labels

Access labels through:
- **Next Actions view** - Shows active tasks with at least one label
- **Task details** - See all labels applied to a task
- **Label filter** - Filter the current task list by a specific label

## Label Colors

Labels maintain their colors from your task provider:
- Todoist label colors are synced automatically
- Microsoft To Do categories show their assigned colors when available

## Filtering by Label

The label dropdown above the task list shows all available labels from your tasks. Select a label to show only tasks with that label applied.

See [Projects](/tasks/features/projects) for organizing tasks into project groups.
