# Dashboard

The Dashboard provides an at-a-glance overview of your task statistics and productivity metrics.

## Statistics Cards

The dashboard displays key metrics including:

- **Total Tasks** - All tasks across your connected accounts
- **Completed Today** - Tasks you've finished today
- **Overdue** - Tasks past their due date
- **Upcoming** - Tasks due in the next 7 days

## Task Views

Switch between different task views using the sidebar:

| View | Description |
|------|-------------|
| Today | Tasks due today or without a due date |
| Overdue | Tasks past their due date |
| Next Actions | Filter tasks by labels |
| All Tasks | Complete list of all tasks |

## Quick Actions

From the dashboard, you can:
- Click any task to view details
- Use the checkbox to mark tasks complete
- Click the + button to add a new task

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New task |
| `Ctrl+F` | Search tasks |
| `Ctrl+,` | Open settings |
