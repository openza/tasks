---

Openza Tasks uses a Windows-native shell with top-level navigation, task commands, a projects pane, a task list, and a task details pane.

## Main Areas

| Area | Purpose |
|------|-------|---------|
| **Navigation** | Switch between task views, Sync, and Settings |
| **Command bar** | Search, add tasks, import/export Markdown, sync, and create projects |
| **Projects pane** | Browse and search projects |
| **Task list** | View, select, complete, copy IDs, and delete tasks |
| **Task details** | Edit the selected task |

## Navigation

The left navigation provides:
- **Next Actions** - Tasks ready to work on
- **Today** - Tasks due today
- **Overdue** - Past-due tasks
- **Tasks** - All tasks
- **Completed** - Finished tasks
- **Sync** - Provider sync status and manual sync
- **Settings** - Appearance, providers, and backups

## Projects Pane

The second pane shows all your projects:

- **Search** - Find projects by name
- **Provider names** - See which provider each project belongs to
- **Project Selection** - Click to filter tasks by project

See [Projects](/tasks/features/projects) for details on managing projects.

## Tasks List

The main content area displays tasks:

- **Task Cards** - Show title, project, due date, and priority
- **Checkboxes** - Mark tasks complete with one click
- **Selection** - Click a task to open details
- **Compact actions** - Copy the task ID or delete the task

### Sorting Tasks

Use the first dropdown above the list to order tasks by:

| Sort Option | Description |
|-------------|-------------|
| Priority | Urgent tasks first, then high, normal, low |
| Due Date | Earliest deadlines first |
| Created Date | Newest tasks first |
| By Project | Group tasks by project |

### Filtering Tasks

Use the priority and label dropdowns above the list to narrow the current task view.

## Task Details

The rightmost pane shows the selected task:

- **Title & Description** - View and edit task content
- **Project Assignment** - Change the task's project
- **Due Date** - Set or modify deadlines
- **Priority** - Set task priority level
- **Labels** - Add or remove labels

## Quick Actions

From any pane, you can:
- Click any task to view its details
- Use checkboxes to mark tasks complete
- Use task row actions for copy/delete
- Use keyboard shortcuts for common actions

## State Persistence

The last selected top-level view and window size are saved locally.
