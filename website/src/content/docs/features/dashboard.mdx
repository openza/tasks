Openza Tasks uses a Windows-native WinUI shell with persistent navigation, task commands, a projects pane, task lists, source intake, sync status, settings, and a details pane.

## Main Areas

| Area | Purpose |
|------|---------|
| **Navigation** | Switch between task views, Inbox review, Sync, and Settings |
| **Command bar** | Add tasks, import/export Markdown, sync, and create projects |
| **Projects pane** | Browse, search, and create Openza projects |
| **Task list** | Search, filter, group, select, complete, copy IDs, and delete tasks |
| **Task details** | Edit the selected task and review source-linked fields |

## Navigation

The left navigation provides:

- **Inbox** - New local tasks and provider items waiting for review
- **Next Actions** - Tasks ready to work on
- **Today** - Tasks due today
- **Overdue** - Past-due tasks
- **Tasks** - All tasks
- **Completed** - Finished tasks
- **Sync** - Provider sync status and manual sync
- **Settings** - Appearance, integrations, spaces, and backups

## Projects Pane

The projects pane shows Openza projects in the selected space:

- **Search** - Find projects by name
- **Project Selection** - Click to filter tasks by project
- **New project** - Create a local Openza project
- **Inbox** - See tasks without a project

See [Projects](/tasks/features/projects) for details on managing projects.

## Tasks List

The main content area displays the current task view:

- **Task rows** - Show title, project, date, priority, labels, and source context
- **Checkboxes** - Mark tasks complete with one click
- **Selection** - Click a task to open details
- **Compact actions** - Copy the task ID or delete eligible local tasks

### Sorting Tasks

Use the sort dropdown above the list to order tasks by:

| Sort Option | Description |
|-------------|-------------|
| Priority | Urgent tasks first, then high, normal, low |
| Date | Earliest dated work first |
| Created Date | Newest tasks first |
| By Project | Group tasks by project |

### Filtering Tasks

Use search, priority, label, and recurring filters above the list to narrow the current task view. Some views also support grouping, such as by date, project, status, priority, label, source, repeating state, created date, or completed date.

## Task Details

The rightmost pane shows the selected task:

- **Title & Description** - View and edit task content
- **Project Assignment** - Change the task's project
- **Date & Deadline** - Set planned dates and deadlines
- **Priority** - Set task priority level
- **Labels** - Add or remove labels
- **Source context** - See provider details for source-linked tasks

## Inbox Review

Provider sync can surface source items in Inbox review before they become Openza task wrappers. You can add useful source items to Inbox, skip items you do not want, and keep provider-owned source data separate from your local planning fields.

## Quick Actions

From any pane, you can:
- Click any task to view its details
- Use checkboxes to mark tasks complete
- Use task row actions for copy/delete
- Use the command bar for add, import, export, and sync

## State Persistence

The last selected view, window size, settings, and backup preferences are saved locally.
