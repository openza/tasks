## Features

<CardGrid stagger>
  <Card title="Windows Native" icon="laptop">
    Built with WinUI 3, Windows App SDK, and MSIX packaging.
  </Card>
  <Card title="Local-First Storage" icon="seti:db">
    Tasks, projects, labels, backups, and imports live on your device first.
  </Card>
  <Card title="Provider Sync" icon="puzzle">
    Reconnect Todoist or Microsoft To Do when you want provider sync.
  </Card>
  <Card title="Focused Task Surface" icon="document">
    Navigation, projects, task list, and details stay visible for daily work.
  </Card>
  <Card title="Backups" icon="seti:config">
    Create and restore durable local SQLite backups, with optional OneDrive snapshots.
  </Card>
  <Card title="Markdown Import & Export" icon="random">
    Import GFM checkboxes and export project-grouped Markdown.
  </Card>
</CardGrid>
