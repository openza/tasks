## Features

<CardGrid stagger>
  <Card title="Local-First Storage" icon="seti:db">
    Your tasks live on your device, always accessible. No account required to get started.
  </Card>
  <Card title="Provider Sync" icon="puzzle">
    Optionally connect Todoist or Microsoft To-Do to sync tasks from the cloud.
  </Card>
  <Card title="4-Pane GTD Layout" icon="laptop">
    Productivity-focused interface with Navigation, Projects, Tasks, and Details panes.
  </Card>
  <Card title="Automatic Backups" icon="document">
    Daily automatic backups keep your data safe. Restore or export anytime.
  </Card>
  <Card title="Dark Theme" icon="moon">
    Beautiful dark theme support for comfortable use in any lighting condition.
  </Card>
  <Card title="Import & Export" icon="random">
    Import tasks from markdown files and export your data for backup or sharing.
  </Card>
