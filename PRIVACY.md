# Privacy Policy

Openza Tasks does not collect telemetry, analytics, or personal usage data.

Task data is stored locally on your Windows device in the app data folder. If you connect Todoist or Microsoft To Do, Openza Tasks uses provider tokens to sync task data with that provider. Todoist tokens and current Microsoft access tokens are stored locally using Windows Credential Locker. The Microsoft MSAL cache is stored locally and encrypted for the current Windows user.

Openza Tasks does not operate a hosted sync service in V1 and does not sell or share data.
